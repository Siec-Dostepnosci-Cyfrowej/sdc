# 6. Informacja końcowa

Okresowa ocena stanu zgodności ma charakter diagnostyczny.

Brak stwierdzonych niezgodności nie oznacza automatycznie pełnej zgodności rozwiązania z wymaganiami WCAG lub EN 301 549.

Wyniki oceny powinny być wykorzystywane do:

- monitorowania stanu dostępności,
- planowania działań naprawczych,
- oceny skuteczności wdrażanych zmian,
- doskonalenia systemu zapewniania dostępności cyfrowej.
