## Kiedy stosować

Procedura uproszczona jest przeznaczona dla małych podmiotów, prostych serwisów, ograniczonego zasobu albo pojedynczych decyzji o wycofaniu treści. Może być stosowana wtedy, gdy decyzja nie dotyczy złożonego systemu teleinformatycznego, dużej migracji danych ani zasobu o wysokim ryzyku prawnym lub organizacyjnym.

Uproszczenie nie oznacza rezygnacji z odpowiedzialności. Podmiot powinien nadal ustalić, czego dotyczy decyzja, jaki jest status zasobu, jaki jest jego status dostępnościowy, czy zasób powinien pozostać dostępny i jak udokumentować rozstrzygnięcie.

## Minimalny przebieg

1. Ustalenie, czego dotyczy zasób.
2. Sprawdzenie, czy zasób powinien pozostać publicznie dostępny.
3. Ocena dostępności cyfrowej zasobu, jeżeli pozostaje publicznie dostępny.
4. Sprawdzenie, czy zasób ma znaczenie archiwalne, prawne, informacyjne lub dostępnościowe.
5. Podjęcie decyzji o pozostawieniu, dostosowaniu, zapewnieniu dostępu alternatywnego, oznaczeniu jako archiwalny, migracji, wycofaniu albo przekazaniu do dalszego postępowania.
6. Udokumentowanie decyzji.
7. Zapewnienie dostępu albo informacji o dalszym trybie udostępnienia, jeżeli jest to potrzebne.

Jeżeli zasób jest migrowany, należy sprawdzić, czy zachowano elementy dostępnościowe. Jeżeli zasób nie spełnia wymagań dostępności cyfrowej, decyzja powinna wskazywać dostosowanie, dostęp alternatywny, oznaczenie albo uzasadnione wycofanie.

Decyzję można udokumentować w krótkim formularzu, rejestrze decyzji, notatce służbowej albo innym narzędziu stosowanym w podmiocie. Ważne jest zachowanie informacji pozwalającej ustalić, co zostało zrobione, kiedy, przez kogo i dlaczego.
