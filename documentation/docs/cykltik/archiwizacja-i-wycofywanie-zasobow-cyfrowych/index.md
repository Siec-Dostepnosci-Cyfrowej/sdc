# Zarządzanie archiwizacją i wycofywaniem zasobów cyfrowych

Zalecenie określa zasady organizacyjnego zarządzania zasobami cyfrowymi w końcowej fazie ich cyklu życia, w szczególności przy archiwizacji, migracji danych, wycofywaniu treści z publikacji oraz wycofywaniu systemów teleinformatycznych.

Pakiet składa się z trzech warstw:

1. właściwego zalecenia - określającego poziom normatywny i strategiczno-organizacyjny,
2. załączników wdrożeniowych - wspierających organizację procesu w podmiotach publicznych,
3. materiałów pomocniczych i operacyjnych - zawierających przykładowe narzędzia do wykorzystania przez podmiot.

Zalecenie nie zastępuje instrukcji kancelaryjnej, jednolitego rzeczowego wykazu akt, instrukcji archiwalnej, decyzji właściwego archiwum państwowego ani analizy prawnej konkretnej sprawy.
