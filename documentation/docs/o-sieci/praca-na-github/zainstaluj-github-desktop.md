## Cel

Zainstalujesz GitHub Desktop i połączysz go ze swoim kontem GitHub.

## Krok po kroku

### 1. Pobierz program

Wejdź na stronę:

https://desktop.github.com

Pobierz wersję odpowiednią dla swojego systemu.

### 2. Uruchom instalator

Otwórz pobrany plik i postępuj zgodnie z instrukcjami instalatora.

### 3. Zaloguj się do GitHuba

Po uruchomieniu programu wybierz:

**Sign in to GitHub.com**

Zaloguj się na konto, którego używasz w pracy Sieci.

### 4. Zezwól na połączenie

GitHub może poprosić o potwierdzenie połączenia z aplikacją GitHub Desktop.

Wybierz **Continue** albo **Authorize**, jeśli taki przycisk się pojawi.

### 5. Zakończ konfigurację

Jeśli program pokaże ekran konfiguracji Gita, zostaw wpisane dane i wybierz **Finish**.

## Co może pójść nie tak

**Program nie widzi mojego konta**

Upewnij się, że logujesz się na właściwe konto GitHub.

**Przeglądarka pyta, czy otworzyć GitHub Desktop**

Wyraź zgodę. To normalny etap łączenia aplikacji z kontem.

**GitHub Desktop pyta o Git**

Jeśli aplikacja proponuje instalację lub konfigurację Gita, zaakceptuj domyślne ustawienia.

## Co dalej

Po instalacji możesz:

- sklonować repozytorium,
- skonfigurować edytor,
- dodać dokument z GitHub Desktop.
