## Cel

Dowiesz się, kiedy warto używać GitHub Desktop i czego potrzebujesz do pracy na swoim komputerze.

## Najprostsza ścieżka

Do pracy w Sieci wystarczy przeglądarka internetowa.  
Możesz dodawać dokumenty, komentować projekty i zgłaszać poprawki bez instalowania dodatkowych programów.

Zacznij od poradników:

- [Dodaj dokument bezpośrednio na GitHubie](./dodaj-dokument-z-kopii-repo)
- [Komentuj i sugeruj poprawki](./komentuj-i-sugeruj-poprawki)

## Kiedy warto użyć GitHub Desktop

GitHub Desktop może być wygodniejszy, gdy:

- często pracujesz nad dokumentami,
- chcesz edytować pliki na swoim komputerze,
- wolisz korzystać z własnego edytora,
- przygotowujesz większe zmiany.

## Czego potrzebujesz

Do pracy na komputerze potrzebujesz:

1. aplikacji **GitHub Desktop**,
2. edytora tekstu lub kodu,
3. lokalnej kopii repozytorium na swoim komputerze.

Nie musisz osobno instalować Gita, jeśli GitHub Desktop działa poprawnie.
