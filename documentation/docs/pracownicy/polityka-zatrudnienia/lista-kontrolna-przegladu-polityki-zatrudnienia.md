## 1. Cel dokumentu

Celem listy kontrolnej jest wspieranie okresowego przeglądu polityki zatrudnienia oraz praktyk kadrowych pod kątem możliwości zatrudniania, rozwoju zawodowego i utrzymania zatrudnienia osób z niepełnosprawnościami.

Lista może być wykorzystywana podczas:

- przeglądu polityki zatrudnienia;
- przeglądu polityki różnorodności;
- oceny warunków pracy;
- planowania działań związanych z dostępnością cyfrową;
- przygotowywania planów doskonalenia organizacji.

---

## 2. Sposób korzystania z listy

Dla każdego pytania należy określić jeden z poniższych statusów:

| Status      | Znaczenie                                            |
| ----------- | ---------------------------------------------------- |
| TAK         | Wymaganie jest spełnione.                            |
| CZĘŚCIOWO   | Działania są realizowane częściowo lub niesystemowo. |
| NIE         | Wymaganie nie jest spełnione.                        |
| NIE DOTYCZY (N/D) | Pytanie nie ma zastosowania do organizacji.          |

W kolumnie „Uwagi i działania” można odnotować obserwacje, ryzyka lub planowane działania doskonalące.

## 13 Wskazówka

Przegląd polityki zatrudnienia warto przeprowadzać łącznie z:

- oceną barier cyfrowych ograniczających zatrudnianie osób z niepełnosprawnościami,
- analizą zgłoszeń pracowników,
- oceną skuteczności dostosowań i racjonalnych usprawnień,
- przeglądem dostępności środowiska pracy,
- oceną realizacji polityki zapewniania dostępności organizacji.

Pozwala to spojrzeć na zatrudnianie osób z niepełnosprawnościami jako element szerszego systemu zarządzania dostępnością i personelem.
