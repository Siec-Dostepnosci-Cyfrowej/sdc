## Informacje dla użytkownika

Za pomocą formularza możesz:

- zgłosić problem związany z dostępnością cyfrową,
- poinformować o trudnościach w korzystaniu ze strony internetowej, aplikacji lub usługi cyfrowej,
- wystąpić o zapewnienie dostępności cyfrowej,
- wystąpić o dostęp alternatywny do informacji lub usługi.

Podanie danych kontaktowych umożliwi przekazanie informacji o sposobie rozpatrzenia sprawy.
