## Cel dokumentu

Lista kontrolna służy do oceny, czy użytkownicy są w stanie łatwo odnaleźć informacje dotyczące dostępności oraz uzyskać potrzebne wsparcie.

W przeciwieństwie do tradycyjnych list kontrolnych narzędzie koncentruje się nie na sprawdzeniu, czy określone informacje zostały opublikowane, lecz na ocenie, czy użytkownik potrafi wykonać konkretne zadania.

Lista może być wykorzystywana:

- podczas przeglądów serwisu internetowego,
- podczas badań z udziałem użytkowników,
- podczas ocen dojrzałości organizacji,
- podczas projektowania lub przebudowy sekcji „Dostępność”.

---

## Instrukcja stosowania

Przy każdym zadaniu odpowiedz:

| Odpowiedź | Znaczenie                                                         |
| --------- | ----------------------------------------------------------------- |
| TAK       | Użytkownik może łatwo znaleźć informację lub wykonać zadanie      |
| CZĘŚCIOWO | Zadanie jest możliwe do wykonania, ale wymaga dodatkowego wysiłku |
| NIE       | Użytkownik nie może znaleźć informacji lub wykonać zadania        |
| ND        | Nie dotyczy organizacji                                           |

Ocena powinna być wykonywana bez wcześniejszej znajomości struktury serwisu.

## Interpretacja wyników

Sekcja „Dostępność” spełnia swoją funkcję tylko wtedy, gdy użytkownik potrafi znaleźć potrzebną informację i wykonać zamierzone zadanie.

Duża liczba odpowiedzi „NIE” lub „CZĘŚCIOWO” wskazuje, że problemem może być nie brak informacji, lecz sposób ich organizacji, nazewnictwo, nawigacja lub język zastosowany w serwisie.

Ocena powinna być traktowana przede wszystkim jako narzędzie identyfikowania barier informacyjnych z perspektywy użytkownika.
