## Cel dokumentu

Lista kontrolna wspiera ocenę kompletności, aktualności i użyteczności sekcji „Dostępność” publikowanej w serwisie internetowym organizacji.

Narzędzie może być wykorzystywane podczas:

- tworzenia sekcji „Dostępność”,
- okresowych przeglądów serwisu internetowego,
- aktualizacji informacji o dostępności,
- przygotowania do audytów i ocen dojrzałości organizacji.

---

Dla każdego pytania zaznacz:

| ----------- | ------------------------------------------- |

| NIE         | Wymaganie nie zostało spełnione             |
| NIE DOTYCZY | Wymaganie nie ma zastosowania               |

### Interpretacja wyników

| Wynik                      | Interpretacja                                                                                |
| -------------------------- | -------------------------------------------------------------------------------------------- |
| 90–100% odpowiedzi TAK     | Sekcja jest kompletna i dobrze wspiera użytkowników.                                         |
| 70–89% odpowiedzi TAK      | Sekcja spełnia większość potrzeb użytkowników, ale wymaga uzupełnień.                        |
| 50–69% odpowiedzi TAK      | Występują istotne luki informacyjne.                                                         |
| poniżej 50% odpowiedzi TAK | Sekcja nie zapewnia użytkownikom skutecznego dostępu do informacji o dostępności i wsparciu. |
