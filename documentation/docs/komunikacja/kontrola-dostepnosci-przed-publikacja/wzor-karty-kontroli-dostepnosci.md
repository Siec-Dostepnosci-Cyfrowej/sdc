# Wzór karty kontroli dostępności przed publikacją

Karta jest przykładowym narzędziem dokumentowania wyniku kontroli. Podmiot może stosować ją jako formularz, tabelę w systemie obiegu zadań, wpis w CMS, arkusz albo część rejestru publikacji.

Zakres wypełniania karty powinien być dostosowany do skali działania podmiotu, rodzaju materiału i ryzyka związanego z publikacją.

| Pole | Informacja do wpisania |
|---|---|
| Tytuł materiału | Nazwa publikowanej treści lub identyfikator sprawy. |
| Typ materiału | Np. PDF, DOCX, XLSX, strona CMS, BIP, formularz, komunikat transakcyjny, newsletter, e-mail masowy, grafika, audio, wideo, post w mediach społecznościowych, treść w systemie wewnętrznym. |
| Kanał publikacji | Np. serwis WWW, BIP, intranet, system wewnętrzny, newsletter, platforma społecznościowa. |
| Osoba przygotowująca | Imię i nazwisko albo rola osoby przygotowującej materiał. |
| Osoba sprawdzająca | Imię i nazwisko albo rola osoby sprawdzającej dostępność. |
| Data kontroli | Data wykonania kontroli. |
| Zastosowana lista kontrolna lub standard | Nazwa listy kontrolnej, standardu publikacji albo innego narzędzia kontroli. |
| Wynik kontroli | Spełnia wymagania / wymaga poprawy / wymaga decyzji warunkowej / publikacja wstrzymana. |
| Decyzja publikacyjna | Publikacja / skierowanie do poprawy / publikacja warunkowa / wstrzymanie publikacji. |
| Uwagi | Krótki opis braków, ryzyk, ograniczeń albo ustaleń. |
| Termin poprawy, jeśli dotyczy | Data i osoba lub komórka odpowiedzialna za poprawę. |
| Tryb publikacji warunkowej, jeśli dotyczy | Uzasadnienie, osoba zatwierdzająca, termin naprawy i sposób monitorowania. |

## Minimalny wariant wpisu

W małych podmiotach albo przy materiałach o niskim ryzyku karta może zostać ograniczona do krótkiego wpisu zawierającego tytuł materiału, typ, kanał, osobę sprawdzającą, datę, wynik kontroli i decyzję publikacyjną.
